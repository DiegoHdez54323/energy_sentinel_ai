# Energy Sentinel Frontend

Aplicacion movil en Expo / React Native para Energy Sentinel.

## Desarrollo

1. Instala dependencias:

```bash
npm install
```

2. Inicia la app:

```bash
npx expo start
```

La app usa `expo-router` y actualmente incluye el flujo inicial de `login`, `register` y una pantalla temporal post-auth mientras se portan las pantallas restantes del template.
